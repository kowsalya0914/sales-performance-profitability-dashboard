# sales-performance-profitability-dashboard

## Project Overview
This Power BI dashboard analyzes sales performance to identify profitability trends, underperforming products, discount impact, and regional performance using the Sample Superstore dataset.

The objective is to provide clear, data-driven insights to support business decision-making and profitability improvement.

---

## Dataset
Sample Superstore Dataset (Kaggle)

Fields Used:
- Order Date
- Region
- Category
- Sub-Category
- Product Name
- Sales
- Profit
- Discount

---

## Tools & Techniques
- Power BI Desktop
- Power Query (Data Cleaning & Transformation)
- DAX Measures
- Profit Margin Analysis
- Discount Impact Analysis
- Regional Performance Evaluation

---

## Dashboard Structure

### Page 1: Sales Performance & Profitability Dashboard
Includes:
- KPI Cards (Total Sales, Total Profit, Profit Margin %)
- Sales & Profit Trend Over Time
- Profit by Discount Bucket
- Profit Margin by Region
- Regional Sales vs Profit Comparison
- Product-Level Profitability Table

---

## Key Business Insights
- Higher discount segments contribute significantly to reduced profitability.
- Certain products generate high revenue but negative profit margins.
- Profitability varies significantly across regions.
- Margin trends reveal performance fluctuations across time periods.
